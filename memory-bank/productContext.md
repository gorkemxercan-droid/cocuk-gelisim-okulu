# Product Context

## Features & Pages

### 1. Home Page (`index.html`)
-   **Hero Section**: Welcoming message with animations.
-   **Info Sections**: Vision, Mission, Values.
-   **Footer**: Quick links and contact info.
-   *Current State*: Uses a blue/pastel theme with "İlk Yaşlar" branding.

### 2. Team Page (`ekibimiz.html`)
-   **Staff Listing**: Grid view of teachers with photos and roles.
-   **Details**: Specific branches (Drama, Sports, Music, Science).
-   *Current State*: Uses a yellow/warm theme with "Renkli Düşler" branding.

### 3. Events/Programs (`etkinlikler.html`)
-   **Workshop List**: Filtering by category (Art, Science, Sports, Music).
-   **Featured Event**: Highlighted "Summer School" or similar.
-   **Newsletter**: Subscription form.
-   *Current State*: Uses a yellow/warm theme with "Renkli Dünyalar" branding.

### 4. Contact (`iletisim.html`)
-   **Contact Form**: Name, Email, Subject, Message.
-   **Map**: Visual representation (image) of the location.
-   **Contact Info**: Address, Phone, Email cards.
-   *Current State*: Uses a yellow/warm theme with "Renkli Anaokulu" branding.

## User Flows
-   **Visitor**: Lands on Home -> Browses Events/Team -> Goes to Contact -> Submits Form.
-   **Parent**: Checks specific workshop details -> Checks Teacher background -> Registers for event.
