# Tech Context

## Technology Stack
-   **HTML5**: Semantic markup.
-   **CSS**: TailwindCSS (v3.x via CDN).
    -   Plugins: `forms`, `typography`, `container-queries`.
-   **JavaScript**: Vanilla JS (ES6+).
-   **Fonts**:
    -   `Quicksand`, `Fredoka` (Google Fonts) - *On Homepage*
    -   `Plus Jakarta Sans` (Google Fonts) - *On Inner Pages*
-   **Icons**:
    -   `Material Icons Round` - *On Homepage*
    -   `Material Symbols Outlined` - *On Inner Pages*

## Development Environment
-   **Local**: Static file serving.
-   **Browser**: Chrome/Firefox/Edge.
